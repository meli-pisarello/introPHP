## MVC - Modelo Vista Controlador

```
# MVC separa el desarrollo del software en capas bien definidas
# Modelo: responsable de la lógica del software y de la conexión con la base de datos.
# Controlador: se encarga de gestionar las peticiones del usuario, las procesa llamando al modelo y luego muestra las respuestas a través de las vistas
# Vistas: muestran al usuario los resultados que se obtienen del modelo a través del controlador.
```
